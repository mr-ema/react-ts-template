# Welcome To WRT

A Webpack + React + Typescript Template.
## Installation

```bash
git clone https://github.com/mr-ema/react-ts-template.git
```



## Run Locally

```bash
npm install
```

```bash
npm run start:dev
```

## Build

```bash
npm run build
```

- You can runned with your custom server or use VsCode Live Server extension
